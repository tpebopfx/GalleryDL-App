# 📥 Загрузчик Медиа (GalleryDL Android)

Мобильное Android-приложение для удобного и массового скачивания изображений с популярных арт-площадок и социальных сетей. Работает на базе мощного движка `gallery-dl`, упакованного в удобный локальный веб-интерфейс.

---

## ✨ Особенности

* **Поддержка множества сайтов:** Danbooru, Pixiv, Twitter, Reddit, Gelbooru, DeviantArt, Instagram и скачивание по прямым ссылкам.
* **Массовая загрузка:** Возможность скачивать медиа целыми пачками по тегам, ID или юзернеймам.
* **Умная сортировка:** Опциональное создание отдельных папок для каждого тега/автора.
* **История скачиваний:** Встроенная база данных SQLite запоминает, что, когда и в каком объеме вы скачивали.
* **Локальный сервер:** Интерфейс работает плавно и быстро прямо на вашем устройстве через Flask.

---

## 🛠 Технологии

Проект написан на **Python** и скомпилирован под Android.
* **Backend:** Python 3, Flask
* **Движок загрузки:** [gallery-dl](https://github.com/mikf/gallery-dl)
* **Сборка APK:** Buildozer, Kivy
* **База данных:** SQLite

---

## 🚀 Установка и запуск (для Android)

1. Скачайте актуальный `.apk` файл из раздела [Releases] (или соберите его самостоятельно через GitHub Actions).
2. Установите приложение на свой Android-смартфон.
3. При первом запуске **обязательно** предоставьте приложению разрешение на доступ к файлам и медиа (иначе загрузка не сработает).
4. Все скачанные файлы по умолчанию сохраняются в папку: `/sdcard/Download/GalleryDL_App`
